## Задача
Реализовать кастомный хук useViewportSize для отслеживания размеров viewport в React-приложениях.
## Описание
Хук рассчитывает высоту и ширину viewport-а, записывает их в состояние(поля width b height) и возвращает состояние. Далее ширина и высота выводятся на страницу
